Na linha 12, o usuário deve inserir, conforme necessário, a lista de IDs de membros da whitelist, ou
seja, membros que não serão expulsos pelo bot.
Na linha 13, o usuário deve inserir o ID do usuário ao qual o bot deve responder. O bot não responderá
aos comandos de ninguém além do detentor de tal ID.
Na última linha de código, é necessário que o usuário informe, entre as aspas simples, o token da aplicação.
